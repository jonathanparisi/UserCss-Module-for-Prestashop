Download this file to your computer. Unzip the folder or find the folder called "UserCss-Module-for-Prestashop". 

1. Change the name on the folder from "UserCss-Module-for-Prestashop" to "usercss"
2. Zip your folder
3. Enter your prestashop backend
4. Enter Modules and add new module
5. Choose the usercss.zip file and upload it
6. Install the module
7. Configurate your module.

This module allows you to cusotmize the layout of the order page in the prestashop backoffice. 
All profiles existing to the users who have an account to the backoffice can have their own layout.

We on mackabler.dk, use it to hide functions that fx our supports do need to see and get confused by.
